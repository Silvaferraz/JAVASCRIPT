  • Aritméticos
    •

- --> Subtração
* --> Multiplicação
/ --> Divisão
% --> Resto da divisão inteira
** --> Potência

-> ()

  • Atribuição
    • 
Auto-atribuições:
var n = 3
n= n+3 -> n, valia 3, agora esta valendo 6, e asssim sucessivamente enquanto utilizarmos estas auto-atribuiçoões
Simplificando:
n += 4 -> somente se a variável recebe ela mesmo + 4

Operador de Identidade: 
Mesmo valor e mesmo tipo, 
ex: 5==='5' --> false, um é number e outro é string
    5===5 --> true
    5!=='5' --> true

  • Ternário
    • 
Opradoor ternário como o próprio nome ja diz possui 3 etapas e é dado assim:
Teste?True:False
Se estabelece oq deseja fazer, dependendo doq foi dito, ele irá mostrar um Valor "True" ou "False"
    Ex: 
            var media = 5
        
        média >= 7?'Aprovado':'Reprovado'
        
        Reprovado
